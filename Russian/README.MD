## The alphabet:

It takes 5 minutes to learn the alphabet. Think of the strange letters as Greek and they'll make sense:

д is delta, thus "d".
г is gamma, thus "g"
р is rho, thus "r"
п is pi, thus "p"
ф if phi, thus "f"
